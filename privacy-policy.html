2.3 Tool-Specific Data Processing

Our tools process data entirely within your browser. The following describes how each tool category handles your data:

Text Processing Tools (Word Counter, etc.)

All text processing occurs locally in your browser. No text data is transmitted to external servers or stored beyond your current session.

Code Generators (QR Code, Barcode, UUID)

Generation happens client-side using JavaScript libraries. Your input data remains on your device.

Conversion Tools (Unit Converter, Time Zone)

All calculations and conversions occur locally. No conversion data leaves your browser.

Image Processing (Thumbnail Creator)

Image processing uses the Fabric.js library and occurs entirely in your browser. Uploaded images are not transmitted to our servers.
